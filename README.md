# Sina's Portfolio

A personal portfolio landing page created as a project for Software Design course with Dr. Keshavarz.

## About

This is a Persian (Farsi) language portfolio website for Sina Sarikhani, featuring:
- Personal introduction
- About section
- Navigation menu
- Social media links
- Responsive design with RTL (right-to-left) layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap Icons
- Custom fonts

## How to View

Simply open `index.html` in a web browser to view the portfolio page.
